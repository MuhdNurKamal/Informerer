# Informerer
Track your weekly [Betterer](https://github.com/phenomnomnominal/betterer) improvements on your Slack channel

## Getting Started

### Set up Virtual environment
```shell
python3 -m venv venv

. ./venv/bin/activate
```

### Install Dependencies
```shell
pip install -r requirements.txt
```

### Run
```
python run.py
```
